Explaining CSS3 hover effects

When I started out as a front-end dev, I always had a hard time understanding how CSS3 hover effects worked, so I created this website that will hopefully make them easier to understand.

The source code for this website is available for download, and I highly recommend doing it so you can read it alongside this website.

Current version - 1.0
This is not a CSS library
If you want a CSS3 hover library, check out Ian Lunn's amazing hover.css library. This website is merely an attempt to explain and make CSS3 hover libraries much easier to understand, and possibly implement. But you can still copy and paste my code into yours I guess.
In the markup example below, it's very obvious that I'm applying the hover effect on an image, but you can replace the img tag with a div tag and the effect will still work: 
<div class="box hover-push">
    <img src="#">
    <div class="mask">
        // put your content here
    </div>
</figure>
Once you've gotten the basic markup down, all that's left to do is to manipulate the effect itself through CSS. I've provided some examples in the gallery below as well - hovering over them will play the respective effects, as well as how to modify them to suit your puposes. You can also download the source code of this website (download button is at the top) and just copy-paste the effect's code into your own document if you need a quick fix.

The best thing about these CSS3 effects is that you can combine various properties together to create your own custom effect!

type 1 - push

Change push direction though the translateX() and/or the translateY() values, or combine them into a single translate() value.

type 2 - slide

Change slide direction though the translateX() and/or the translateY() values, or combine them into a single translate() value.

type 3 - fade

Control fade timing with the transition-timing property, or combine it into the transition property.

type 4 - old-school spin

Change number of spins through the rotate() value.

type 5 - zoom in

The scale() value starts at 0, then increases to 1 on hover.

type 6 - zoom out

The scale() value starts at any number larger than 1, then decreases to 1 on hover.

type 7 - side hinge

Change hinge direction with the various transform-origin properties, and hinge direction though the rotate() values.

type 8 - corner hinge

Change hinge corner with the transform-origin property, and hinge direction through the rotate() value.

type 9 - shutter out

Use scaleX() for horizontal shutter, scaleY() vertical shutter.

type 10 - shutter in

Use scaleX() for horizontal shutter, scaleY() vertical shutter.

type 11 - flip

Change flip direction with the transform-origin property, and though the rotate() values.

type 12 - outline

Change outline colour with the outline-colour property, timing with the transition property. (This effect works better if there is an existing border)
